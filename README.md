# 📑 Практична робота №4: Адаптивна картка статті з інтерактивними елементами

## 📝 Опис проекту

Цей проект присвячений створенню адаптивної та семантично коректної картки статті (Article Preview Component) за наданим макетом. Основна увага приділена реалізації адаптивності для різних типів пристроїв (Mobile, Tablet, Desktop) та створенню інтерактивної панелі поширення (share) за допомогою чистих технологій CSS.

Картка реалізована як семантичний компонент `<article>`, що забезпечує гнучкість макета та високу якість коду.

## 🚀 Технології

- **HTML5** (семантичні теги: `<article>`, `<h2>`, `<p>`, `<small>`)
- **CSS3** (Custom Properties, Flexbox, Media Queries)
- **БЕМ (Block Element Modifier)** — для структурованості та незалежності стилів (блок `.article`).
- **Google Fonts** — використано шрифт **Manrope** (weights 500, 700).
- **Font Awesome** — для іконок соціальних мереж та кнопки поширення.

## ✨ Особливості реалізації

- **Адаптивність (Mobile First):**
  - **Mobile (< 768px):** Вертикальна структура. При активації кнопки "Share" панель соцмереж динамічно перекриває область автора.
  - **Tablet (768px - 1439px):** Перехід до горизонтальної орієнтації. Зображення та контент розташовані поруч у співвідношенні 40/60.
  - **Desktop (1440px):** Оптимізований вигляд для великих екранів із фіксованою максимальною шириною.
- **Інтерактивність:** Реалізовано "спливаюче вікно" (tooltip) для десктопної версії за допомогою стану `:focus` та селекторів сиблінгів, що дозволяє інтерфейсу бути живим без обов'язкового використання JavaScript.
- **Семантика:** Використано правильну ієрархію заголовків та групування метаданих автора, що відповідає кращим практикам верстки.
- **Дизайн:** Реалізовано характерні закруглені кути (`border-radius: 12px`), тіні (`box-shadow`) та повну відповідність кольоровій гамі зі Style Guide.
